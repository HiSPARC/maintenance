Introduction
============

This documentation describes the steps required to solve problems with an |hisparc| station.  It first explains how to recognize different problems and then how to solve them.

The first part, :doc:`faq`, might help answer some questions that you have.

The second part, :doc:`known-issues`, describes some of the known issues that we have encountered with the HiSPARC software and hardware. It discribes how to determine if the problem described is what you are experiencing and then possible solutions.
